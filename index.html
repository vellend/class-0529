Canadian Space Agency Case Study

Research & Testing

1) A research plan was created with the intent to solidify a methodology, and create a questionnaire. 2) Five usability tests were conducted. The users were recorded while they performed the action: looking for a job at the Canadian Space Agency, getting back to the main page from the jobs board and finally, looking for current research. Testing the current CSA navigation was performed through user testing. 3) Two proto personas were developed in order to define the majority of users who would be using this site.

Results

Ideation Phase

The two users both grouped specific types of people (kids, teachers, post secondary, and future employees) as separate categories. Both users grouped sub headers related to learning together. Users had two different headers, “Other” and “News & Events”. The grouping was mostly a category for getting in contact, social media and games for children. The LATCH principles that were used included: categorization which created tertiary sub headers, and hierarchy for a clear call to action on every page.

Results

Wireframing & Prototype Phase

Wireframe- Top Navigation Menu The wireframe was designed to have an emphasis on implementing exciting signifiers with cards on the main page. The main landing page has the exact menu options that the top navigation has. The Canadian Armed Forces website was the inspiration for the resulting top navigation wireframe layout and the main page layout.
